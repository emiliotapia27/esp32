ESP32 - Modelos y Proyectos

Introducción

La ESP32 es una familia de microcontroladores con Wi-Fi y Bluetooth integrados, diseñados para aplicaciones de IoT y embebidas.

Modelos Destacados

  • ESP32-WROOM-32: Versión estándar con buena conectividad Wi-Fi y Bluetooth.
  • ESP32-CAM: Incluye soporte para cámara, ideal para proyectos de videovigilancia.
  • ESP32-S3: Enfocado en inteligencia artificial y computación de alto rendimiento.
  • ESP32-PICO-D4: Versión compacta con componentes integrados para diseños pequeños.

Características Generales

  1. Procesador dual-core a 240 MHz.
  2. Conectividad Wi-Fi 2.4 GHz y Bluetooth 4.2/5.0.
  3. Múltiples interfaces: UART, SPI, I2C, ADC, DAC, PWM.
  4. Bajo consumo energético con diferentes modos de suspensión.

Proyectos Posibles

Proyecto 1: Cámara de Seguridad con ESP32-CAM

Este proyecto permite transmitir video en tiempo real mediante Wi-Fi.

Proyecto 2: Control de Luces con ESP32 y Bluetooth

Permite encender y apagar luces a través de una aplicación móvil usando Bluetooth.
